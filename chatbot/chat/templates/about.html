About
This project is created by Glison Pereira.

The Python Chatbot is a chatbot that provides relevant code snippet based on the related Python question asked. This chatbot acts like a small-scale retrieval-based system. The technology used to create this bot is Django, NLP, and Machine Learning, and SQLite 3 serving as a database. The algorithm uses cosine similarity to find the top 3 closest questions using n_neighbors from the dataset and return the corresponding code snippets. Out of which the 1st one with the maximum similarity is chosen. The Dataset used is Dataset.
